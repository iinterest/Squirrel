LESS 混合

什么是『混合』：

混合（Mixins）是 LESS 中的一种特殊方法，它可以将一个定义好的 class A 轻松的引入到另一个 class B 中，从而简单实现 class B 继承 class A 中的所有属性；还可以带参数地调用，就像使用函数一样。—— lesscss.net

混合样式：

Squirrel 的混合样式存放在 base.less 文件中，在浏览器兼容性方面，默认只支持 Webkit 内核，如果需要支持其它内核浏览器可以到『定制』页面进行定制操作。

文字尺寸设置：

@head-font-size:18px;
@title-font-size:16px;
@content-font-size:14px;
@description-font-size:12px;
@badge-font-size:10px;

盒模型：

.border-box()

.demo {.border-box;}
.display-box()

.demo {.display-box();}
.box-orient()

.demo {.box-orient();}
.box-pack()

.demo {.box-pack;}
.box-align()

.demo {.box-align;}
.box-flex()

.demo {.box-flex;}

圆角：

.demo {.rounded-corners();}

阴影：

.demo {.shadow();}

变形：

.demo {.transition();}

绝对布局平铺：

.demo {.overspread (@top: 0, @right: 0, @bottom: 0, @left: 0);}

形状：

.demo1 {.sq-arr-up (@size, @color)}
.demo2 {.sq-arr-right (30px, #d7d7d7)}
.demo3 {.sq-arr-down (@size, @color)}
.demo4 {.sq-arr-left (@size, @color)}
