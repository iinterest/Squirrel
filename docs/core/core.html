SQ.core

对 Javascript 进行基本的功能扩展，增加类型判断支持、面向对象支持等。

命名空间

命名空间方法

SQ.core.namespace("SQ.modules.module2");

面向对象

面向对象的常用方法，如继承、

SQ.core.namespace("SQ.modules.module2");

类型判断

.isString

判断是否为字符串

SQ.core.isString("squirrel");  // return true
.isArray

判断是否为数组

SQ.core.isArray([1,2,3]);  // return true
.isNumber

判断是否为数字

SQ.core.isNumber(23);  // return true
.isBoolean

判断是否为布尔值

SQ.core.isBoolean(true);  // return true
.isNull

判断是否为 Null

SQ.core.isNull(null);  // return true
.isUndefined

判断是否为 undefined

SQ.core.isUndefined(undefined);  // return true
.isFunction

判断是否为函数

function fn() {
    alert("this is a function");
}
SQ.core.isFunction(fn);  // return true
.isObject

判断是否为对象

SQ.core.isObject({name: "squirrel"});  // return true

SQ.dom

用于存储常用的 jQuery Dom 元素，在具体的业务中也可以将页面 jQuery Dom 存入 SQ.Dom 对象。

SQ.dom.$demo = $(".demo");

已封装的节点包括：

SQ.dom.$win;    //$(window)
SQ.dom.$doc;    //$(document)
SQ.dom.$body;   //$("body")

SQ.store

前端数据持久化，支持对 cookie、localStorage、sessionStorage 进行增删改操作，支持 applicationCache 操作。

Cookie 操作

实现对 Cookie 的增删改操作。

.cookie.get()
SQ.store.cookie.get("name");           // 读取
.cookie.set()
SQ.store.cookie.set("name", "value");  // 设置

"name", "value" 是必要实参，更多可选实参包括：有效期(秒)、路径、域、加密安全设置；
其中有效期除可设置为day(一天有效期，第二天 0 点 0 分过期)、week(一周有效期)。

SQ.store.cookie.set("name", "value", "day", "/");
.cookie.del()
SQ.store.cookie.del("name");           // 删除

SQ.ua

返回一个包含设备系统版本、浏览器名称及版本等信息的对象，具体包括：

SQ.ua.os                 // 操作系统信息
SQ.ua.os.name            // 操作系统名称，android || ios
SQ.ua.os.version         // 操作系统版本，例如：4.2、5.1
SQ.ua.os.device          // 设备信息，ipod || iphone || ipad
//----------------------------------------------------------------------
SQ.ua.browser.core       // 浏览器内核，webkit
SQ.ua.browser.version    // 浏览器版本，例如：159.2、7.9、8.x(UC 浏览器 8.6、8.7 等)
SQ.ua.browser.shell      // 浏览器壳信息，chrome || safari || opera || ucweb
SQ.ua.browser.module     // 浏览器模式，swift(UC 浏览器急速模式，目前只有国内 Android UCWeb 9.0+ 可判断急速模式)

SQ.util

生成函数

可以生成随机数、随机数组、唯一标识符等。

.generate.randomInt(min, max)

从最小与最大至之间返回一个随机数。

var randomNum = SQ.util.generate.randomInt(1, 10);      // return 3
.generate.randomArr(arr)

将一个数组随机排列后返回。

var arr = [1,2,3,4,5]
var randomNum = SQ.util.generate.randomArr(arr);      // [2,3,5,1,3]

.goTop()

回到页面顶部。

.goBack()

返回上一页。
