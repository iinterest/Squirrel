动画

引用 Animate.css 动画样式并精选了 26 个常用的。如果需要更多可以使用『定制』功能。

示例：

Animate Demo

DOM 使用：

可以为 DOM 元素添加动画基础样式animated加特殊动画样式bounceIn来实现动画效果，例如：

<class="sq-btn animated bounceIn"></a>
插件调用：

除了直接使用动画样式，还可以在支持动画的插件中设置ANIMATE来使用，例如 SQ.Popup 插件：

var popupDemo = new SQ.Popup({
    DOM_TRIGGER_TARGET: ".J_popupDemo",
    CSS_CLASS: ".popup-demo",
    CSS_WIDTH: 300,
    CSS_HEIGHT: 150,
    VERTICAL: "middle",
    HORIZONTAL: "center",
    ANIMATE: ".fadeIn"
});
