图标(Deprecated)

v0.8.24 版本后不推荐使用，请使用 Font Icon 代替；v0.8.25 版本将移除 sq-icon。

图标.sq-icon，可以方便的添加至按钮、工具栏的按钮组、导航或表单等各种模块上，名为 sq-icon@2x.png 的图片即为全部图标的 Sprite，制作时单个的标准尺寸为 40 × 40 像素。

示例：

HTML 代码：

<i class="sq-icon home"></i>
<i class="sq-icon msg"></i>
<i class="sq-icon up"></i>

图标样式：

home msg up download update
good refresh search arrow
magic star speaker heart
magic-w download-w search-w

为按钮添加图标：

home
msg
up
download
<div class="sq-btn f-grey"><i class="sq-icon home"></i>home</div>
