SQ.Dialog (Deprecated)

v0.8.24 版本后不推荐使用，请使用 SQ.Popup 代替；v0.8.25 版本将移除 SQ.Dialog。

SQ.Dialog 是一个弹出对话框组件，可以异步加载内容在窗口内显示。

示例：

Dialog Demo

如何使用：

加载：

初次使用时，可以在定制页面勾选 Dialog 组件，然后下载 Squirrel 代码。

如果是在已有工程中新增，可以将 dialog.js、dialog.less 文件分别加入到工程目录的src/js、src/less 文件夹下，Squirrel 框架构建工具会自动进行合并处理。然后加入相应的 HTML 和 JS 代码：

HTML 代码：
<div class="J_showDialog">Show Dialog</div>
JS 代码：
var dialogDemo1 = new SQ.Dialog({
    EVE_EVENT_TYPE : "click",
    DOM_TRIGGER_TARGET : ".J_showDialog",
    TXT_CLOSE_VAL : "×",
    CSS_STYLE : "demo1",
    FULLSCREEN_OFFSET : [20, 20],
    ANIMATE : ".bounceIn",
    MASK : true,
    LOCK : true,
    show : function (e) {
        var me = this;
        me.$dialogContent.append("<p class='demo-text'>全屏窗口示例</p>");
    }
});

参数：

Name Type Description
config object 组件配置（下面的参数为配置项，配置会写入属性）
Properties
Name Type Description
EVE_EVENT_TYPE string 绑定事件设置
DOM_TRIGGER_TARGET string 被绑定事件的 Dom 元素
TXT_CLOSE_VAL string 关闭按钮显示文字
TXT_OK_VAL string 确定按钮显示文字，默认值："确定"
TXT_CANCEL_VAL string 取消按钮显示文字，默认值："取消"
CSS_STYLE string 对话框样式名称
CSS_MASK_BACKGROUND string 遮罩背景色，默认值："#000000"
CSS_MASK_OPACITY string 遮罩透明度，默认值：0.5
CSS_POSITION string 对话框定位方式，默认值："absolute"
CSS_TOP number 对话框 top 属性值，默认值："auto"
CSS_LEFT number 对话框 left 属性值，默认值："auto"
CSS_WIDTH number 对话框 width 属性值，默认值："auto"
CSS_HEIGHT number 对话框 height 属性值，默认值："auto"
CSS_MIN_WIDTH number 对话框 min-width 属性值
CSS_MIN_HEIGHT number 对话框 min-height 属性值
FULLSCREEN_OFFSET array 全屏偏移距离设置，即弹出层距上下左右的距离 例如：[20,10]表示上下距离 20 像素，左右距离 10 像素
NUM_CLOSE_TIME number 对话框自动关闭时间，单位：毫秒
VERTICAL string 对话框垂直位置，可选值："middle"， 必须设置 CSS_HEIGHT 或 CSS_MIN_HEIGHT 才能生效
HORIZONTAL string 对话框水平位置，可选值："center"， 必须设置 CSS_WIDTH 或 CSS_MIN_WIDTH 才能生效
ANIMATE string 动画类，默认值：undefined
MASK boole 遮罩设定，默认为 false，设为 true 将显示遮罩效果
LOCK boole 锁定操作，默认为 false，设为 true 将屏蔽触摸操作，默认值：false
DESTROY boole 销毁操作，设为 true 时将会在 dialog 首次显示后清除目标 DOM 的绑定事件。
NUM_CLOSE_TIME number 自动关闭对话框时间，单位：毫秒
PREVENT_DEFAULT boole 默认动作响应设置，默认为 true，不响应默认操作
DELAY number 延时显示对话框设置，单位：毫秒
show function 打开对话框回调函数
ok function 点击确定按钮回调函数
cancel function 点击取消按钮回调函数
close function 关闭对话框回调函数

内部文档对象：

SQ.Dialog 内封装了以下 jQuery(zepto) 对象，可以在回调函数中直接使用：

var me = this;
me.$dialogWindow   //主对话框窗口
me.$dialogContent  //对话框内容显示区，异步加载时可将内容填充至该区域。
me.$okBtn          //确定按钮
me.$cancelBtn      //取消按钮
me.$close          //关闭按钮

内部数据：

var me = this;
me.top          //对话框 top 数据
me.left         //对话框 left 数据
me.width        //对话框 width 数据
me.height       //对话框 height 数据

回调函数：

SQ.Dialog 内置了以下回调函数，可以在触发相应事件时调用：

var dialogDemo = new SQ.Dialog({
    EVE_EVENT_TYPE : "click",
    DOM_TRIGGER_TARGET : ".J_showDialog",
    TXT_CLOSE_VAL : "×",
    CSS_STYLE : "demo3",
    CSS_WIDTH : 240,
    CSS_HEIGHT : 100,
    VERTICAL : "middle",
    HORIZONTAL : "center",
    NUM_CLOSE_TIME : 2000,
    show : function () {
        // 显示对话框时执行
    },
    ok : function () {
        // 点击确定按钮时执行
    },
    cancel : function () {
        // 点击取消按钮时执行
    },
    close : function () {
        // 点击关闭按钮时执行
    }
});
