如何使用：

加载：

初次使用时，可以在定制页面勾选 Tabs 组件，然后下载 Squirrel 代码。

如果是在已有工程中新增，可以将 tabs.js、tabs.less 文件分别加入到工程目录的src/js、src/less 文件夹下，Squirrel 框架构建工具会自动进行合并处理。然后加入相应的 HTML 和 JS 代码：

HTML 代码：
<div class="dpl-tabs J_tabs">
    <ul class="tabs col3">
        <li>tab 1</li>
        <li>tab 2</li>
        <li>tab 3</li>
    </ul>
    <div class="panels">
        <p>Proin elit arcu, ...</p>
    </div>
    <div class="panels">
        <p>Morbi tincidunt, ...</p>
    </div>
    <div class="panels">
        <p>Duis cursus ...</p>
    </div>
</div>
JS 代码：
var tabs = new SQ.Tabs({
    EVE_EVENT_TYPE : "mouseover",
    DOM_TRIGGER_TARGET : ".J_tabs",
    DOM_TABS : ".tabs>li",
    DOM_PANELS : ".panels",
    NUM_ACTIVE : 0,
    show : function () {

    }
});

参数

Name Type Description
config object 组件配置（下面的参数为配置项，配置会写入属性）
Properties
Name Type Description
EVE_EVENT_TYPE string 触发事件，click 或 mouseover
DOM_TRIGGER_TARGET string 被绑定事件的 Dom 元素
DOM_TABS string 标签 Dom 元素
DOM_PANELS string 面板 Dom 元素
API_URL string API 接口① 字符串形式
API_URL array API 接口② 数组形式，数组中各项对应各个选项卡
CSS_HIGHLIGHT string 自定义高亮样式名称，默认为 active
CSS_LOADING_TIP string loading 提示样式
TXT_LOADING_TIP string loading 提示文字
NUM_ACTIVE number 初始高亮选项卡序号，0 - n
NUM_XHR_TIMEER number XHR 超时时间
CLEAR_PANEL boolean 切换选项卡时是否自动清理面板数据
LOCAL_DATA string Ajax 数据 loaclstorage 开关，默认为 false
NUM_EXPIRES number Ajax 数据 loaclstorage 过期时间（单位：分钟），默认为 15 分钟
trigger function 触发选项卡切换回调函数
show function 显示选项卡时回调函数
beforeLoad function 异步加载前回调函数
loaded function 异步加载成功回调函数

内部文档对象：

SQ.Tabs 内封装了以下 jQuery(zepto) 对象，可以在回调函数中直接使用：

var me = this;
me.$triggerTarget           //输入框
me.$tabs    //联想词显示面板
me.$panel        //清空输入框按钮
me.$activeTab    //联想词显示面板
me.$activePanel    //联想词显示面板

内部数据：

var me = this;
me.activeIndex    //联想词显示面板

回调函数：

SQ.Suggest 内置了以下回调函数，可以在触发相应事件时调用：

var tabs = new SQ.Tabs({
    EVE_EVENT_TYPE : "mouseover",
    DOM_TRIGGER_TARGET : ".J_tabs",
    DOM_TABS : ".tabs>li",
    DOM_PANELS : ".panels",
    trigger : function (ds) {
        // 触发选项卡切换回调函数
    },
    show : function (ds) {
        // 显示选项卡时回调函数
    },
    beforeLoad : function (ds) {
        // 异步加载前回调函数
    },
    loaded : function (ds) {
        // 异步加载成功回调函
    }
});
